# **App Name**: Contratei.Online

## Core Features:

- Candidate Profile: Allow candidates to create and edit profiles with skills, experience, and salary expectations.
- Company Profile: Enable companies to register and manage their profiles.
- Job Posting: Allow companies to post detailed job descriptions.
- AI-Powered Resume Analysis: Use the Google Gemini API to automatically analyze resumes, identify keywords, and determine compatibility with job postings.  This will function as a tool.
- Personalized Job Recommendations: Provide candidates with personalized job recommendations based on their experience, skills, and preferences using AI.
- Intelligent Candidate Filtering: Help companies filter candidates based on AI-driven compatibility scores and keyword analysis.
- Real-time Notifications: Send real-time notifications via Firebase Cloud Messaging to keep users informed.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF), representing trust and professionalism, fitting for a recruitment platform.
- Background color: Light grayish-blue (#E6F3FF), offering a clean and unobtrusive backdrop.
- Accent color: Turquoise (#40E0D0) for interactive elements and highlights, adding a touch of vibrancy.
- Body text and headlines: 'Inter', a sans-serif font offering a neutral and modern feel.
- Use consistent, professional icons to represent job categories and actions.
- Maintain a clean, responsive layout optimized for both desktop and mobile devices.
- Use subtle animations to enhance user experience, such as loading indicators and transitions.
